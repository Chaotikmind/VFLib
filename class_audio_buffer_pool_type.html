Detailed Description

template<class TypeOfCriticalSectionToUse = CriticalSection>
class AudioBufferPoolType< TypeOfCriticalSectionToUse >

Template for AudioBufferPool with a specified lock type.

This provides an instance of AudioBufferPool using the specified type as the lock. The default is to use a CriticalSection. Alternatively, a DummyCriticalSection may be used when the caller is responsible for synchronization.

Request a temporary buffer.

The pool returns a buffer with the specified number of channels and enough room for at least numSamples. The caller owns the buffer until it is released with a matching call to releaseBuffer().

Parameters:
numChannelsThe number of channels requested.
numSamplesThe number of samples per channel requested.

The resulting buffer.
