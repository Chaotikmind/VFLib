Detailed Description

template<class Element, class Tag = LockFreeQueueDefaultTag>
class vf::LockFreeQueue< Element, Tag >

Multiple Producer, Single Consumer (MPSC) intrusive FIFO.

This container uses the same intrusive interface as List. It is wait-free for producers and lock-free for consumers. The caller is responsible for preventing the ABA problem (http://en.wikipedia.org/wiki/ABA_problem)

Invariants:

  • Any thread may call push_back() at any time (Multiple Producer).
  • Only one thread may call try_pop_front() at a time (Single Consumer)
  • The queue is signaled if there are one or more elements.
Parameters:
TagA type name used to distinguish lists and nodes, for putting objects in multiple lists. If this parameter is omitted, the default tag is used.

bool try_pop_front ( Element **  pElem)

Attempt to retrieve an element.

This attempts to pop an element from the front of the list. The return value indicates if the operation was successful. An operation is successful if there is no contention for the list. On a successful operation, an element is returned if the list was non empty, else nullptr is returned. On failure, the returned element is undefined.

This operation is wait-free.

Parameters:
[out]pElemThe element that was retrieved, or nullptr if the list was empty.
