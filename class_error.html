Detailed Description

This lightweight but flexible class records lets you record the file and line where a recoverable error occurred, along with some optional human readable text.

A recoverable error can be passed along and turned into a non recoverable error by throwing the object: it's derivation from std::exception is fully compliant with the C++ exception interface.
