Detailed Description

template<class SchemaType>
class SortedLookupTable< SchemaType >

Sorted map for fast lookups.

This container is optimized for a data set with fixed elements.

SchemaType obeys this concept:

struct SchemaType
{
typename KeyType;
typename ValueType;
// Retrieve the key for a specified value.
KeyType getKey (Value const& value);
};

To use the table, reserve space with reserveSpaceForValues() if the number of elements is known ahead of time. Then, call insert() for all the your elements. Call prepareForLookups() once then call lookupValueByKey ()
