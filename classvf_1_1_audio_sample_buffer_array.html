Detailed Description

template<int Channels = 2>
class vf::AudioSampleBufferArray< Channels >

A wrapper for an array of pointers to audio data.

This simple class wraps an array of pointers to multi-channel audio data. Provided functions can move the pointers back and forth as a group, making certain forms of code that manipulate audio buffers more concise.

Conversion to and from AudioSampleBuffer allow seamless usage.
