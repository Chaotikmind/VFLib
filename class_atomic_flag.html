Detailed Description

A thread safe flag.

This provides a simplified interface to an atomic integer suitable for representing a flag. The flag is signaled when on, else it is considered reset.

bool AtomicFlag::trySignal ( )

Signal the flag.

If two or more threads simultaneously attempt to signal the flag, only one will receive a true return value.

Returns

void AtomicFlag::signal ( )
noexcept

Signal the flag.

The flag must be in the reset state. Only one thread may call this at a time.

void AtomicFlag::reset ( )
noexcept

Reset the flag.

The flag must be in the signaled state. Only one thread may call this at a time. Usually it is the thread that was successful in a previous call to trySignal().
