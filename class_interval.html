Detailed Description

template<class Ty>
class Interval< Ty >

A half-open interval.

This represents the half-open interval [begin, end) over the scalar type of template parameter Ty. It may also be considered as the specification of a subset of a 1-dimensional Euclidean space.

Template Parameters:
TyA scalar numerical type.
