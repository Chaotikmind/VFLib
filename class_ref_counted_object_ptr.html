Detailed Description

template<class ReferenceCountedObjectClass>
class RefCountedObjectPtr< ReferenceCountedObjectClass >

The template parameter specifies the class of the object you want to point to - the easiest way to make a class reference-countable is to simply make it inherit from RefCountedObjectType, but if you need to, you could roll your own reference-countable class by implementing a pair of mathods called incReferenceCount() and decReferenceCount().

When using this class, you'll probably want to create a typedef to abbreviate the full templated name - e.g.

typedef RefCountedObjectPtr <MyClass> MyClassPtr;
