Detailed Description

These are reference counted, so multiple ref objects may point to the same item in the registry. When the last ref is deleted, the registry reference is unrefed (via luaL_unref).

Note:
The implementation of the reference counting is not thread safe, since this would require C++11 or platform-specifics. This should not be a problem, since a lua_State is generally not thread safe either.
